Bienvenu(e)!

Objectif:

Eliminez les enemies pour remplir votre bar de JS. Une fois que vous la remplissez, une partie du CV sera débloqué et vous passez au niveau suivant.

Completez le CV pour gagner!

Instructions:

Utilisez les touches Q et D pour vous déplacer à gauche et à droite. Ciblez avec la souris et tirez avec click gauche. Vous pouvez pauser le jeu avec la touche P

Les enemies peuvent donner des boosts de HTML et de CSS si vous récupérez les logos avec votre vaiseeau.

Si un enemie touche la ligne rouge ou si votre niveau de HTML est finit, c'est game over!

A propos:

Ce jeu a été développez dans le cadre de ma formation de développeur FullStack JS à IFOCOP. Il s'agit de mon premier projet Javascript abouti !

J'ai utilisé que du code Javascript "pur" pour le réaliser, sans aucune librairie ou framework.
